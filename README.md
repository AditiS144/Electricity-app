# Electricity-app

This project is a React-based Electricity Bill Management System that allows users to log in, view their electricity bills, and analyze usage through charts. It includes dynamic bill calculation with tax, a detailed breakdown feature, and a simulated payment system with confirmation. The UI is enhanced using modern design techniques like glassmorphism and supports both dark and light modes.


Tech stack used:
React → for UI & logic 
JavaScript → for calculations & interactivity 
CSS → for styling 
Chart.js → for graph
